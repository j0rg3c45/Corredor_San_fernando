# San Fernando ITT

Proyecto de análisis y estandarización de datos para el Instituto Tecnológico de Tlajomulco - Corredor San Fernando.

## Estructura del proyecto

```
San_Fernando_ITT/
├── .kiro/steering/       → Reglas de estilo y contexto para el agente
├── Agente/               → Rol y lineamientos del Data Steward
├── Data/                 → Esquema estandarizado y documentación de datos
├── Notebooks/            → Notebooks de análisis
└── Outputs/              → Mapas HTML exportados y otros resultados
```

## Uso

1. Consultar `Agente/contexto_agente.md` para entender el rol del Data Steward.
2. Revisar `Data/esquema_datos_comunes.txt` para el esquema propuesto.
3. Ejecutar los notebooks en `Notebooks/` para el análisis.
4. Los resultados se exportan a `Outputs/`.
